Termux+Hexo+Github搭建网站

2022-02-11

使用Termux+Hexo+Github在手机上免费搭建个人网站……

注：

  1. 请多用翻译
  2. 如果遇到错误，请多使用Google
  3. 请使用科学上网

Termux

安装Termux

  1. 打开Termux下载地址
  2. 找到最新版下载
  3. 安装

配置

  1. 打开Termux
  2. 更新配置
  3. 安装相关组件
    依次输入下列指令
    安装Vim 安装Nodejs 安装Git 安装Openssh 更新Npm

部署

  1. 安装Hexo插件
  2. 生成Hexo文件
  3. 进入hexo
  4. 修改文件 进入编辑模式 更改文末为 退出
  5. 安装部署插件

秘钥

  1. 返回根目录
  2. 配置信息
  3. 生成秘钥 复制这串字符，保存起来

Github

账号

  1. 打开官网
  2. 按照相关提示注册账号

    仓库

  3. 新建仓库，仓库名为用户名.github.io
  4. 进入SSH设置页面，选择New SSH key，名称任意，内容为刚刚得到的字符串
  5. 保存

    上传

  6. 回到Termux
  7. 进入网站目录
  8. 上传 中途会询问你，选择yes
  9. 完成

最后

等待一会后在你的网站就好了，网址是你的用户名.github.io
